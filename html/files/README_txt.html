Nicovideo

Nicovideo is scraping NICONICO DOUGA utility for Ruby (NICONICO DOUGA is Japanese Video Sharing Service)

  • NICONICO DOUGA : www.nicovideo.jp/

Installation

 $ sudo gem install nicovideo

Usage

See sample/nv_download.rb
